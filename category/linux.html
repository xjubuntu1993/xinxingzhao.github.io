linux添加命令别名

添加命令别名是一件很爽的事情

对于一些每天都要输入很多便的命令，给建个别名肯定是再好不过了

alias ..="cd .."
alias ..2="cd ../.."
alias ..3="cd ../../.."
alias ..4="cd ../../../.."
alias ..5="cd ../../../../.."

alias gadd="git add ."
alias gcommit="git commit -am "
alias gpush="git push origin master"
alias gpull="git pull origin master"
